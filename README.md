# Job Matching Frontend

Фронтенд приложение для платформы подбора вакансий и соискателей с использованием ML-модели.

## Возможности

- ✅ Создание вакансий работодателями
- ✅ Создание профилей соискателями
- ✅ Автоматическое определение профессий с помощью ML-модели
- ✅ Поиск подходящих совпадений (matches)
- ✅ Фильтрация вакансий и профилей по профессии и местоположению
- ✅ Современный и отзывчивый UI

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- Axios
- Tailwind CSS

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Убедитесь, что бэкенд API запущен на `http://localhost:8000`

3. Запустите dev-сервер:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`

## Использование

### Для работодателей

1. Перейдите на страницу "Создать вакансию"
2. Заполните форму с описанием требований
3. ML-модель автоматически определит подходящую профессию
4. Просмотрите подходящих соискателей на странице деталей вакансии

### Для соискателей

1. Перейдите на страницу "Создать профиль"
2. Заполните форму с описанием навыков и опыта
3. ML-модель автоматически определит подходящую профессию
4. Просмотрите подходящие вакансии на странице деталей профиля

## API Endpoints

Приложение использует следующие эндпоинты:

- `POST /api/vacancies` - Создание вакансии
- `GET /api/vacancies` - Список вакансий
- `GET /api/vacancies/:id` - Детали вакансии
- `GET /api/vacancies/:id/matches` - Совпадения для вакансии
- `POST /api/profiles` - Создание профиля
- `GET /api/profiles` - Список профилей
- `GET /api/profiles/:id` - Детали профиля
- `GET /api/profiles/:id/matches` - Совпадения для профиля

Подробная документация API доступна на `http://localhost:8000/docs`
